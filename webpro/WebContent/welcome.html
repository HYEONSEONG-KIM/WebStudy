html의 태그에 스타일을 부여하기 위해서 css가 필요하다
내부 css - html문서안에  <style>
외부 css - 별도의 css문서를 작성해서 html문서 내부에 파일로 삽입 시킨다
<link rel = "stylesheet" href = "./css/outstyle.css">
